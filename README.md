## 📚 기술 스택

- **Minimum SDK Version** 23

- **Target SDK Version** 31

- **Clean Architecture**  지향

- **Architecture**
    - MVVM

- **Language**
    - 100% Kotlin

- **사용한 라이브러리**
    - Jetpack AAC, Hilt, Room, Retrofit, Gson, Glide, Kakao 로그인 등

- **비동기 처리**
    - Coroutine Flow
    

## ✨ 개발 과정 및 향후 계획
